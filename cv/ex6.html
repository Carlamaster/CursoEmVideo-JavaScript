DOM - Document Object Model - Aula 9

Todo arvoré DOM começa pela a raiz que é window dentro de window tem varios objetos como por exemplo:
location: qual a localização do site, history: o historico.
Em document: contem o HTML(chamado de parent, ou pai),
que contém dois filhos (ou child) o Head e o body
e assism por diante como uma arvoré de parentes mostrando como esta o codigo.

E usado para adicionar ou acessar a arvoré pelo javaScript.

Como utilizar os métodos getElementByTagName, getElementById, getElemenstByName, getElementByClassName, querySelector !

Podemos acessoar por 5 tipos: Por Marca usando o getElementByTagName() para utilizar com mais de uma tag, na forma arrays[0,1,...]
Por ID usando o getElementById() usando a tag e adicionando o elemento id posso chamar apenas a tag relacionada ao id.
Por Nome usando getElemenstByName()
Por Classe usando getElementsByClassName()
Por Seletor usando querySelector
